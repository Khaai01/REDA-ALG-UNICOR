software

El software es un conjunto de programas, instrucciones y datos que permiten que una computadora realice tareas específicas. Incluye desde sistemas operativos y aplicaciones, hasta utilidades y programas de propósito especial. Es la parte lógica y no tangible de un sistema informático, contrastando con el hardware, que es la parte física. El software proporciona las herramientas y la funcionalidad necesarias para que el hardware ejecute diversas operaciones y cumpla con distintas funciones.
